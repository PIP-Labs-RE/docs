title: 합의 계층 (CL)
icon: handshake-simple
---

**합의 계층(Consensus Layer, CL)**은 Cosmos SDK와 CometBFT를 기반으로 구축됩니다. Cosmos SDK는 블록체인 애플리케이션을 구축하기 위한 모듈형 프레임워크를 제공하여 새로운 모듈과 기능을 원활하게 통합하고, 네트워크를 손쉽게 확장 및 커스터마이징할 수 있게 해줍니다. `story` 클라이언트는 Engine API 통합과 새로운 스테이킹 메커니즘을 지원하기 위한 업그레이드 및 추가 Cosmos SDK 모듈을 도입합니다. CometBFT는 고성능, 확장 가능, 보안성을 갖춘 합의 엔진으로, Cosmos 생태계에서 광범위하게 검증되었습니다. CometBFT와 Cosmos SDK는 ABCI++ 인터페이스(ABCI++ 스펙 링크)를 통해 통신합니다. 코드는 [story](https://github.com/piplabs/story) 저장소에서 확인할 수 있습니다.
