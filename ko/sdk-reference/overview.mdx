## Protocol SDK

Protocol SDK (`@story-protocol/core-sdk`)는 DATA Foundation의 IP 관리 계층과 상호작용하기 위한 기본 SDK입니다. IP 자산 등록, 라이선스 조건 첨부, 라이선스 토큰 발행, 로열티 관리, 분쟁 제기 등의 작업에 사용하세요.

## CDR SDK

CDR SDK (`@piplabs/cdr-sdk`)는 DATA Foundation의 Confidential Data Rails
시스템을 위한 TypeScript 클라이언트를 제공합니다. 현재 릴리스는 Aeneid를 대상으로 하며
온체인 시크릿 볼트(`uploadCDR` / `accessCDR`)와 암호화된 파일
워크플로(`uploadFile` / `downloadFile`)를 모두 지원합니다. 이 패키지는 npm에
`@piplabs/cdr-sdk`로 게시되어 있습니다.
