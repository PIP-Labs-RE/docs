**DATA Foundation**은 *멀티 코어* 실행 환경으로 강화된 목적 지향형 탈중앙화 블록체인입니다.
그 아키텍처는 메인 실행 코어와 함께 고도로 커스터마이즈된 여러 코어들로 구성됩니다. 메인 코어는 EVM 동등성을 제공하여 생태계의 기존 애플리케이션을 빠르게 도입할 수 있게 합니다.
특화된 코어 중 하나인 지식재산권(IP) 코어는 지식재산권 등록을 네이티브 자산 클래스로 효율적으로 처리하는 동시에, 복잡한 IP 관계 그래프 전반에 걸친 연산을 최적화합니다. 이 코어는 지능(intelligence)을 프로그래머블한 IP 자산으로 변환합니다. DATA Foundation은 주로 지식재산권에 초점을 맞추고 있지만, 유연한 아키텍처 덕분에 IP 관련 애플리케이션을 훨씬 넘어서 확장될 수 있는 미래의 코어들을 도입할 수 있습니다.


아키텍처에 대한 더 자세한 내용은 [백서](https://www.datafdn.org/whitepaper.pdf)에서 확인할 수 있으며, 토큰 경제 설계에 대해서는 DATA Foundation의 [스테이킹 설계](/network/learn/token-economy) 문서를 참고하세요. 구현에 대한 자세한 내용은 [노드 소프트웨어](/network/learn/node-software) 챕터를 참고하세요.
